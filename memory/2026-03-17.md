# 2026-03-17 - MikeGuard 开发日志

## 🛡️ MikeGuard 安全护栏系统

### 已完成的功能

1. **ML 模型集成**
   - 当前使用：ProtectAI/deberta-v3-base-prompt-injection-v2
   - 准确率：95%+
   - 用途：Prompt 注入检测
   - 配置位置：`~/.openclaw/workspace/MikeGuard/src/mikeguard.py`

2. **Web 管理界面**
   - URL: http://127.0.0.1:8766
   - 功能：
     - 风险阈值配置
     - ML 模型配置
     - 风险模式编辑
     - 拦截日志查看
     - **快速模型切换** (新增)

3. **模型切换功能** (新增)
   - 位置：管理页面 "快速模型切换" 区域
   - 支持模型：
     - ProtectAI v2 (推荐) ✅
     - ProtectAI v1
     - Vijil mBERT ⚠️ (有 tokenizer 问题)
     - ModernBERT (需要认证)
   - API: `POST /api/model/switch`

4. **日志优化**
   - Hook 只记录拦截，不记录放行
   - 服务只记录拦截，不记录放行
   - 内容预览包含用户输入
   - 无重复图标问题

### 🔧 多层保护机制

1. **第 1 层** - OpenClaw Gateway (消息接收)
2. **第 2 层** - MikeGuard Hook (首次检查)
3. **第 3 层** - MikeGuard 服务 (详细检测)
   - 模式匹配 (权重 0.6)
   - ML 模型 (权重 0.4)
4. **第 4 层** - AI 内置安全 (最后防线)
