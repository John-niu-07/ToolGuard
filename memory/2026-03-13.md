## 🔧 Technical Notes

### Environment Variables for ClawGuard
```bash
ENABLE_CLAWGUARD=true
CLAWGUARD_API_URL=http://localhost:8000
CLAWGUARD_TIMEOUT_SECONDS=5
CLAWGUARD_BLOCK_ON_ERROR=true
CLAWGUARD_CHECK_RESPONSES=false
```

### Key Lesson
Open WebUI middleware logging requires `print()` statements (not `log.info()`) because logging isn't initialized when middleware code executes.

### Streaming Response Format
Proper SSE format for Open WebUI:
```
data: {"id": "...", "object": "chat.completion.chunk", ...}\n\n
data: [DONE]\n
```
